# Prompt Format

Use natural language with optional structured tokens.

## Recommended structure
```
<theme or reference>, <floors>, <bays>, <rooms>, <exterior>
```

## Examples
- `Fire station, 2 floors, 4 bays, chief office, dispatch, dorms, kitchen, apron`
- `Hospital, 3 floors, lobby, training area, dorms`

## Supported keywords
### Floors and bays
- `2 floors`, `1 floor`
- `4 bays`, `0 bays`

### Rooms
- chief office
- dispatch
- kitchen
- dining
- gym / workout
- turnout
- classroom / meeting
- wash bay
- dorms
- captain quarters
- lt quarters
- lobby
- fire pole
- training area

### Exterior
- watch tower
- apron
- flag pole

### Style (currently metadata only)
- modern
- industrial
- brick
- stucco
- metal
- concrete
