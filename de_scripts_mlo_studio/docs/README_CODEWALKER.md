# CodeWalker Placement Steps

1. Start CodeWalker and open your GTA V project.
2. Add the generated resource to your FiveM resources folder.
3. Open the `stream/<resource_name>_PLACEME.ymap` file in CodeWalker.
4. Select the anchor prop at the origin (0,0,0).
5. Move/rotate the build to its final location.
6. Export the final YMAP and replace the stub in `stream/`.
